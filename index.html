Тема урока

Продвинутый CSS

Решение

1.1 Создана структура проекта. Внутри проекта создана папка blocks, для хранения блоков по методологии БЭМ. Создана базовая HTML-структура по БЭМ.
1.2 В HTML структуру добавлен блок "Галерея изображений" используя методологию БЭМ.

2.1 Подключен CSS. Элементы стилизованы, используя современные подходы к CSS, такие как Flexbox или Grid
2.2 Изменена стилизация навигации, теперь она располагается вертикально, засчет использования Flexbox и свойства flex-direction: column; Для создания двухстолбцовой структуры контента использовался Grid.

3.1 Создан репозиторий на GitHub, инициализирован локальный репозиторий и связан с удалённым. Получена ссылка на GitHub для просмотре результата.
3.2 Внесено изменение в проект, опубликовано, проверено, что изменения отобразились корректно.

4.1 Страница открыта в нескольких браузерах и отображается корректно. При разработке использовался DevTools для проверки адаптивности страницы на разных устройствах.
4.2 Страница адаптируется под различные размеры экрана, ошибки в отображении исправлены.
